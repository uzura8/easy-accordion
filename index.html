Getting statrted

  • Set class js-accordionto the accordion trigger element.
  • Set attribute data-target="{{target dom id}}"to the trigger element.
  • Set {{target dom id}} to the accordion content block's id.
  • is-active class is added trigger and content block at open.
  • Please define styles for them.
